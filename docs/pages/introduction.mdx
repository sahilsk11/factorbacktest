# Introduction

Welcome to the Factor Backtest API — a quantitative finance platform for backtesting investment factor strategies.

## Overview

Factor Backtest allows you to:
- Define and test quantitative factor expressions against historical market data
- Run portfolio backtests with configurable rebalancing intervals
- Evaluate strategies with performance metrics (Sharpe ratio, annualized return, etc.)
- Execute trades on connected brokerage accounts

## API Base

The API is deployed as an AWS Lambda function behind API Gateway. All endpoints are prefixed with the base URL.

## Getting Started

1. Create an account or sign in
2. Construct a factor expression (e.g., `SMA(close, 20) > SMA(close, 50)`)
3. Run a backtest over your desired date range
4. Analyze results and refine your strategy

## Rate Limits

Public endpoints are rate-limited to 100 requests/minute per IP. Authenticated endpoints have higher limits.
