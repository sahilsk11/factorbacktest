# Supabase Project Deleted Troubleshooting

If you don't use the project for 90 days, the supabase credentials get deleted.

I have a video on my mac called `2025-09-21 01-47-45.mkv`. It has all steps, except you need to update the anon key:

** Project settings > API Keys > Legacy, anon, service_role API keys **

copy that into .env.local
